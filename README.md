# Running

File [Doctest](Doctest.hs) contains all tests for these modules. You can run them using [doctest](https://hackage.haskell.org/package/doctest):

```
$ doctest Doctests.hs
Examples: 26  Tried: 26  Errors: 0  Failures: 0
```
